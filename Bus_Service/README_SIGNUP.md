🐛 TROUBLESHOOTING
============================================================

### "I don't see Sign Up button"
Solution:
1. Rebuild solution (Ctrl+Shift+B)
2. Close and reopen application
3. Check if button appears now

### "Sign Up button doesn't work"
Solution:
1. Make sure you're on Login form
2. Click exactly on the button text
3. Rebuild and try again

### "Registration form won't open"
Solution:
1. Check database connection
2. Verify Database1 exists
3. Try again

### "Get error during signup"
Solution:
1. Read the error message
2. Check SIGNUP_GUIDE.md troubleshooting
3. Try different username if duplicate
4. Fill all required fields

### "Can't login after signup"
Solution:
1. Use exact username from signup
2. Use exact password from signup
3. Check username exists in database
4. Try again

For more help:
→ Read SIGNUP_GUIDE.md (Troubleshooting section)
