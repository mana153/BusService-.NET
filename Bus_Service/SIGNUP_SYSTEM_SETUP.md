STUDENT SIGNUP (STEP BY STEP)
============================================================

### Step 1: Click "Sign Up" from Login Form
- Located below the Login button
- Opens the registration form

### Step 2: Enter Details
- **Username**: Unique name (required)
  Example: "john_doe", "student123"

- **Password**: Your login password (required)
  Example: "SecurePass123"

- **Role**: Select "Student" from dropdown (required)

- **Department**: Select from dropdown (required)
  Available departments:
  - SCIENCE
  - BBA
  - B.COM
  - LAW
  - MBA
  - MSC - DS

### Step 3: Click "Register"
- System validates all fields
- Checks if username is unique
- Creates user in database
- Shows "Registration Successful!"
- Auto-closes registration form
- Returns to Login form

### Step 4: Login
- Use your new username and password
- Select Student role automatically from database
- Access Student Dashboard

============================================================
VOLUNTEER SIGNUP (STEP BY STEP)
============================================================

### Step 1: Click "Sign Up" from Login Form
- Located below the Login button
- Opens the registration form

### Step 2: Enter Initial Details
- **Register Number**: Your unique ID (required)
  Example: "VOL-001", "volunteer-mana"

- **Password**: Your login password (required)
  Example: "SecurePass123"

- **Role**: Select "Volunteer" from dropdown (required)

### Step 3: Generate OTP
- Click **"Generate OTP"** button
- System generates a random 4-digit code
- Dialog shows: "Your OTP is: XXXX"
- Write down or remember the OTP

### Step 4: Enter OTP
- Go to **"OTP"** field
- Enter the exact 4-digit OTP from the dialog
- Example: If dialog shows "5682", enter "5682"

### Step 5: Click "Register"
- System validates:
  - All fields filled
  - OTP matches
  - Register number is unique
- Creates user in Users table
- Creates entry in AllowedVolunteers table
- Shows "Registration Successful!"
- Auto-closes registration form
- Returns to Login form

### Step 6: Login
- Use your register number as username
- Use your password
- Select Volunteer role automatically from database
- Access Volunteer Form

FORM FEATURES AND BUTTONS

VALIDATION RULES
============================================================

### General Validation:
✓ Username: Required, must be unique, min 1 character
✓ Password: Required, min 1 character (no restrictions yet)
✓ Role: Required, must select Student or Volunteer
✗ Admin: Cannot self-register (admin-only)

### Student Specific:
✓ Department: Required, must select from list
✓ No OTP needed
✓ Can register immediately after filling form

### Volunteer Specific:
✓ OTP: Required, must match generated OTP exactly
✓ OTP: Generated by clicking button first
✓ Must verify OTP before registration completes
✓ Creates entry in AllowedVolunteers table
