## ✅ WHAT GETS CHECKED

✓ Username is unique
✓ All required fields filled
✓ Role is valid (Student/Volunteer)
✓ Department selected (for Students)
✓ OTP matches (for Volunteers)
✓ Password is entered

## ❌ COMMON ERRORS

"User already exists"
→ Choose different username

"Please fill all required fields"
→ Fill Username, Password, Role, Department (if Student)

"Department is required"
→ Select Student role first

"Invalid OTP"
→ Enter exact OTP from dialog

"Please generate OTP first"
→ Click Generate OTP button
